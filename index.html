Linux pentru Educaţie

„Rădăcinile educației sunt amare, dar fructul este dulce.”- Aristotel

Linux pentru Educaţie este un sit dedicat promovării sistemelor de operare cu licență liberă pentru sistemul educaţional.

Sunt multe sisteme de operare care au fost create special pentru sistemul educațional, din păcate nu toate mai sunt active și dezvoltate. Mai jos este o listă numai cu acele sisteme de operare care mai sunt încă dezvoltate și întreținute.

  • AcademiX - este un proiect românesc dezvoltat special pentru activități didactice și conține o multitudine de programe cu licențe libere pentru învățământ, începând cu clasele primare și terminând cu învățământul universitar.
  • Scilinux - este o versiune dezvoltată împreună de Fermilab, CERN și diverse alte laboratoare și universități din întreaga lume gata pregătită pentru experimente.
  • Edubuntu - este dezvoltat pentru a aduna cele mai bune programe libere disponibil în educație.
  • DoudouLinux - este dezvoltat special pentru a face utilizarea cât mai simplă și mai plăcută a calculatorului pentru toți copii din lume.
  • DebianEdu/Skolelinux - a fost dezvoltat ca o soluție liberă pentru resursele și nevoile școlilor.
  • openSUSE-Edu Li-f-e - este o distribuție atent creată pentru studenți, profesori și păriniți, în așa fel încât nu mai este nevoie să se instaleze nimic suplimentar.
  • ubermix - este dezvoltat de profesori pentru elevi dar și pentru cadrele didactice, ubermix ia toată complexitatea din dispozitive pentru studenți.
  • Trisquel Sugar - este o distribuție dezvoltată pentru copii cu vârste între 0 și 12 ani.
  • SuliX - este o distribuție maghiară cu bazele pe Fedora, dezvoltată special pentru sistemul educațional.
  • Sugar on a Stick - este un sistem bazat pe Fedora oferind o platformă Sugar premium pentru învățare și proiectat pentru a încăpea pe o memorie USB.
  • ASRI Édu - este o soluție la cheie, eficientă și rapidă potrivit pentru copii cu vârsta între 3 și 12 ani, profesori și începători.
  • Ubuntued - este o ediție educativă a Ubuntu creată de Rudra Saraswat, aceeași persoană care a creat Ubuntu unity și care a fost concepută ca un substitut pentru distribuția care nu se mai dezvoltă, Edubuntu.

  • ALT Linux School - este un sistem de operare personalizat special pentru nevoile instituțiilor de învățământ.
  • Ubuntu EducationPack - este un sistem pentru educație, știință, programare, precum și pentru organizarea procesului de învățământ.
  • LliureX - este un proiect al Consiliului de Cultură, Educație și Sport de la Primăria din Valencia, Spania.
  • UberStudent - este un sistem de operare și tot o dată și o suită de programe pentru toți, în special pentru învățământul superior.
  • Karoshi - este o distribuție creată special pentru server-ele din unitățile de învățământ.
  • OLPC OS - este o inițiativă pentru calculatoare ieftine, cu un sistem de operare pre-instalat și aplicații concepute pentru copii din țările în curs de dezvoltare.
  • Fedora Security Lab - oferă un mediu de test sigur pentru a lucra la auditul de securitate, criminalistică, sistem de salvare și a metodologiilor de testare a securității de predare în universități și alte organizații.
  • Fedora Design Suite - este o distribuție creată pentru designeri și pentru mediul de producție și publicare multimedia.
  • Fedora Robotics Suite - este o distribuție dedicată pentru cei care le place să lucreze în domeniul robotici, precum și cercetătorilor experimentați în robotică.
  • Fedora Scientific - reunește toate instrumentele științifice, de culegere, de scris, de publicarea și numerice sub licență liberă.
  • Fedora Jam - este pentru amatorii de audio și muzicieni care doresc să creeze, editeze și să producă audio și muzică pe GNU+Linux.
  • Zorin OS Educational - a fost construit pentru mediul educațional, astfel că include aplicații pentru învățarea științei, geografiei, matematicii, limbiilor străine și alte subiecte.
